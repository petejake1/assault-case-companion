## 🚀 Deploy in 60 Seconds (No Terminal, No Code)

1. Go to **[vercel.com](https://vercel.com)** and sign in with GitHub (one click).
2. Click **“Add New…” → Project**.
3. Import the repository **`petejake1/assault-case-companion`**.
4. Leave all settings default (Next.js is auto-detected).
5. Click **Deploy**.
6. In ~45 seconds you get a live URL like `https://assault-case-companion-xxx.vercel.app`.
7. Bookmark it. Share the preview link. Done.

That is the entire process. Zero terminal. Zero local setup. Zero configuration.
